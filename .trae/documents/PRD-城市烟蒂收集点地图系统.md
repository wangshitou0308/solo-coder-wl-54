
# 城市公共烟蒂收集点与吸烟区地图系统 - 产品需求文档

## 1. 产品概述

城市公共烟蒂收集点与吸烟区地图系统是一款面向城市管理的公共服务应用，旨在解决城市烟蒂乱扔问题，提升环卫管理效率。系统为市民提供吸烟区与烟蒂收集点的位置查找、状态查看与问题反馈功能，同时为环卫管理方提供设施监控与调度视图。

- **目标用户**：市民（普通使用者）、保洁员（一线作业人员）、管理员（环卫管理人员）
- **核心价值**：减少烟蒂乱扔现象，提升环卫作业效率，数据驱动城市管理决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 市民 | 游客模式/快速登录 | 浏览地图、查询设施、上报问题、追踪工单 |
| 保洁员 | 账号登录 | 接单清理、更新设施状态、查看工单列表 |
| 管理员 | 账号登录 | 全局监控、数据看板、工单调度、设施管理 |

### 2.2 功能模块

1. **首页地图**：全市吸烟区与烟蒂收集点分布地图，多类型点位标注，设施详情卡片
2. **筛选搜索**：按区域、状态、类型筛选，按距离排序查找最近设施
3. **问题上报**：市民上报设施问题，工单生成与进度追踪
4. **数据看板**：设施分布密度、满溢率趋势、响应时效、热力图可视化
5. **角色切换**：市民/保洁员/管理员三种角色视图切换

| 页面名称 | 模块名称 | 功能描述 |
|---------|----------|----------|

| 首页地图 | 点位标记层 | 三种设施类型图标，四级健康度颜色编码 |
| 首页地图 | 设施详情卡片 | 设施编号、类型、容量状态、清理时间、人流密度 |
| 首页地图 | 筛选面板 | 区域筛选、状态筛选、类型筛选、距离排序 |

| 问题上报 | 工单追踪 | 工单列表、处理进度、历史记录 |

## 3. 核心流程

### 3.1 市民使用流程

市民打开应用 → 查看地图上的设施分布 → 使用筛选条件查找目标设施 → 点击查看设施详情 → 如发现问题点击上报 → 填写上报表单 → 提交生成工单 → 追踪工单处理进度

```mermaid
flowchart LR
    A["打开应用"] --> B["浏览地图"]
    B --> C["筛选设施"]
    C --> D["查看详情"]
    D --> E{"发现问题?"}
    E -->|是| F["填写上报表单"]
    F --> G["提交工单"]
    G --> H["追踪处理进度"]
    E -->|否| I["完成使用"]
```

### 3.2 保洁员工作流程

保洁员登录 → 查看待处理工单列表 → 接单 → 前往现场处理 → 更新设施状态 → 完成工单 → 查看下一个工单

```mermaid
flowchart LR
    A["保洁员登录"] --> B["查看待处理工单"]
    B --> C["接单"]
    C --> D["现场处理"]
    D --> E["更新设施状态"]
    E --> F["完成工单"]
    F --> G["继续下一单"]
```

### 3.3 管理员监控流程

管理员登录 → 查看全局监控总览 → 查看数据看板分析趋势 → 查看告警列表 → 调度保洁资源 → 管理设施信息

```mermaid
flowchart LR
    A["管理员登录"] --> B["监控总览"]
    B --> C["数据看板分析"]
    C --> D["查看告警"]
    D --> E["调度保洁资源"]
    E --> F["设施管理"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：烟灰系列（深灰 #2C3E50 作为主色，配合不同层次的灰色调）
- **健康度色阶**：绿色 #27AE60（正常）、黄色 #F1C40F（半满）、橙色 #E67E22（将满）、红色 #E74C3C（已满/告警）
- **辅助色**：蓝色 #3498DB（交互元素）、浅灰 #ECF0F1（背景）
- **整体风格**：简约专业的城市管理风格，清晰的信息层级，数据可视化突出
- **按钮风格**：圆角矩形按钮，悬停微缩放效果，点击反馈
- **字体**：主标题使用思源黑体 Bold，正文使用思源黑体 Regular，数字使用等宽字体
- **布局风格**：卡片式布局，左侧边栏导航 + 主内容区 + 右侧详情面板
- **图标风格**：线性图标 + 填充图标结合，统一描边粗细

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|----------|---------|
| 首页地图 | 顶部导航栏 | Logo、角色切换、搜索框、功能入口 |
| 首页地图 | 地图主区域 | 全屏地图、缩放控件、图层切换 |
| 首页地图 | 左侧筛选面板 | 可折叠筛选面板，多维度筛选条件 |
| 首页地图 | 点位标记 | 自定义图标，健康度颜色外环，脉冲动画（告警点） |
| 首页地图 | 详情弹窗 | 浮层卡片，设施信息列表，操作按钮 |
| 数据看板 | 顶部统计卡片 | 四个核心指标卡片，带趋势箭头 |
| 数据看板 | 图表区域 | 两列布局的图表网格，ECharts 可视化 |
| 数据看板 | 时间选择器 | 日/周/月切换，日期范围选择 |
| 问题上报 | 表单页面 | 分步表单，进度指示，图片上传预览 |
| 问题上报 | 工单列表 | 列表 + 状态标签，时间线进度展示 |

### 4.3 响应式设计

- **设计优先**：桌面端优先设计（1920px 基准）
- **平板适配**：1024px 断点，侧边栏可收起，图表单列布局
- **手机适配**：768px 断点，底部导航栏，全屏地图，筛选底部抽屉
- **触控优化**：按钮最小 44px 触控区域，滑动手势支持

### 4.4 动画与交互

- **页面加载**：骨架屏渐入，内容错峰显示
- **点位动画**：告警点位脉冲呼吸效果，新上报点位弹跳入场
- **卡片交互**：悬停微微上浮 + 阴影加深，点击缩放反馈
- **筛选过渡**：面板滑入滑出，筛选条件淡入淡出
- **图表动效**：数据加载时柱状图/折线图绘制动画
- **角色切换**：页面横向滑动过渡，配色主题微调
